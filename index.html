題目：物體由靜止出發 (\(v_0 = 0\))，以等加速度 \(a\) 跑完位移 \(\Delta x = s\)。求此過程的平均速度 \(\bar{v}_a\)。

依據平均速度的核心定義 \(\bar{v}_a = \frac{\Delta x}{\Delta t} = \frac{s}{t}\)，本題已知位移 \(s\)，但哪一個公式中必須的關鍵物理量尚未給出，需要我們先把它導出來？

✅ 答對了！若要回歸基本定義式計算，必須先用已知量 \((v_0, a, s)\) 導出時間 \(t\)。

步驟二：選定公式求出時間 \(t\)

已知初速 \(v_0 = 0\)、等加速度 \(a\)、位移 \(s\)，應選擇哪一個公式來求解時間 \(t\)？

✅ 完全正確！透過位移公式（此處即 \(s = \frac{1}{2}at^2\)），即可順利將時間 \(t\) 導出。

系統化雙軌解法演示

1. 方法一：依據「平均速度定義」計算（講義代數推導演示）：

  • 根據平均速度基本定義式：\( \bar{v}_a = \frac{\Delta x}{\Delta t} = \frac{s}{t} \)
  • 由位移公式移項求得時間 \(t\)：
    \( s = \frac{1}{2}at^2 \implies t^2 = \frac{2s}{a} \implies t = \sqrt{\frac{2s}{a}} \)
  • 將求得的 \(t\) 帶回定義式分母，進行有理化推導：
    \( \bar{v}_a = \frac{s}{\sqrt{\frac{2s}{a}}} = s \cdot \sqrt{\frac{a}{2s}} = \sqrt{\frac{a \cdot s^2}{2s}} = \sqrt{\frac{as}{2}} \)
2. 方法二：利用等加速度特有性質（公式法對照）：
  • 先用無時間公式求出末速 \(v\)：\( v^2 = v_0^2 + 2as \implies v = \sqrt{2as} \)
  • 再直接代入等加直線運動特有的平均速度公式：
    \( \bar{v}_a = \frac{v_0 + v}{2} = \frac{0 + \sqrt{2as}}{2} = \sqrt{\frac{2as}{4}} = \sqrt{\frac{as}{2}} \)

雙軌對照詳解

1. 公式解 (純代數)：

  1. 全程求 \(a\)：\( x = \frac{1}{2}at^2 \implies a = \frac{2x}{t^2} \)
  2. 無時間公式求新速度：\( v_{\text{new}}^2 = 0^2 + 2a(\frac{x}{3}) \)
  3. 代入 \(a\)：\( v_{\text{new}}^2 = 2(\frac{2x}{t^2})(\frac{x}{3}) = \frac{4x^2}{3t^2} \implies v_{\text{new}} = \frac{2x}{\sqrt{3}t} = \frac{2\sqrt{3}x}{3t} \)
2. v-t 圖解 (利用相似三角形面積比例)：⭐️ 秒殺思維

小三角形(局部)與大三角形(全程)相似。面積比等於高(速度)的平方比：

\( \frac{\text{小面積}}{\text{大面積}} = \frac{x/3}{x} = \frac{1}{3} = \frac{v_{\text{new}}^2}{v_{\text{max}}^2} \implies v_{\text{new}} = \frac{v_{\text{max}}}{\sqrt{3}} \)

由全程面積可知 \( v_{\text{max}} = \frac{2x}{t} \)。代回得 \( v_{\text{new}} = \frac{2\sqrt{3}x}{3t} \)

公式解詳解

設全車長度為 \( L \)，等加速度為 \( a \)。

  1. 觀察全程 (位移 \( \Delta x = L \))：
    \( (3v)^2 = v^2 + 2aL \implies 9v^2 = v^2 + 2aL \implies aL = 4v^2 \)
  2. 觀察半程 (求中點速度 \( v_{\text{mid}} \)，位移 \( \Delta x = \frac{L}{2} \))：
    \( v_{\text{mid}}^2 = v^2 + 2a(\frac{L}{2}) = v^2 + aL \)
  3. 代入求解：
    將 \( aL = 4v^2 \) 代入：\( v_{\text{mid}}^2 = v^2 + 4v^2 = 5v^2 \implies v_{\text{mid}} = \sqrt{5}v \)

公式解詳解

設全車長度為 \( L \)，等加速度為 \( a \)。

  1. 觀察全程 (車頭到車尾通過，位移 \( \Delta x = L \))：
    初速為 \(v\)，末速為 \(3v\)，代入公式：
    \( (3v)^2 = v^2 + 2aL \implies 9v^2 = v^2 + 2aL \implies 2aL = 8v^2 \implies aL = 4v^2 \)
  2. 觀察前 \(1/4\) 程 (位移 \( \Delta x = \frac{L}{4} \))：
    設此時火車通過 P 點的瞬時速度為 \( v_{1/4} \)：
    \( v_{1/4}^2 = v^2 + 2a(\frac{L}{4}) \implies v_{1/4}^2 = v^2 + \frac{1}{2}(aL) \)
  3. 將全程關係式帶入求解：
    將第一步得到的整體常數 \( aL = 4v^2 \) 替換代入：
    \( v_{1/4}^2 = v^2 + \frac{1}{2}(4v^2) = v^2 + 2v^2 = 3v^2 \)

    等號兩邊同時開根號，求得驚奇完美的答案：
    \( v_{1/4} = \sqrt{3}v \)
